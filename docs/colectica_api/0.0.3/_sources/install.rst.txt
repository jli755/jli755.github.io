Installing
==========

Basic installation
------------------

You can install Colectica-API using ``pip``::

    pip install colectica_api


Getting the source code
-----------------------

If you want to get the latest development code, or contribute to this project,
you can get the source code by typing::

    git clone https://github.com/CLOSER-Cohorts/colectica_api
